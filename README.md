# CV Maker (PHP OOP - MVC)

CV maker is a cv builder project with PHP OOP using MVC Design Pattern to generate CV/Resume.

### Knowleadge

- Apache setting (.htaccess)
- Mysql

### Using PHP functions

- call_user_func_array
- extract
- trim
- compact
- explode
- unset
- class_exists
- method_exists
- array_values

### Installation

- git clone
- config `config/app.php`

```
// Application
define('APP_URL', 'http://localhost/project-folder/pubic');

// database
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', '');
define('DB_NAME', 'cv_maker');
```

- composer install
- import database.sql
- entry point `public/index.php`
